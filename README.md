# 🌿 Banana Leaf Disease Classification System

An advanced, full-stack agritech web application built to predict and classify diseases in banana leaves using environmental and physical metrics. Powered by Machine Learning and Django, featuring a premium glassmorphic user interface.

## 🚀 Features
* **Machine Learning Engine:** Trains and evaluates 4 distinct algorithms (Logistic Regression, Decision Tree, Random Forest, K-Nearest Neighbors) to find the highest-accuracy model.
* **Predictive Dashboard:** An asynchronous, single-page application (SPA) interface where users can input 9 distinct plant metrics to receive instant, liquid-animated AI predictions.
* **Secure Authentication:** Built-in user registration, login, and session management systems.
* **Premium UI/UX:** Features a modern, Awwwards-style glassmorphic design utilizing Tailwind CSS, custom scroll-reveals, and fluid loading states.

## 🛠️ Technology Stack
**Backend:**
* Python 3.12+
* Django 5.x
* Scikit-learn (Machine Learning)
* Pandas & NumPy (Data Processing)
* Joblib (Model Serialization)

**Frontend:**
* HTML5
* Tailwind CSS
* Vanilla JavaScript (AJAX & Animations)

## 📋 Dataset Architecture
The model evaluates the following 9 environmental features to determine the health of the plant:
1. Leaf Length (cm)
2. Leaf Width (cm)
3. Color Intensity (Light, Medium, Dark)
4. Spots Present (Binary)
5. Moisture Level (%)
6. Texture (Smooth, Rough)
7. Humidity (%)
8. Temperature (°C)
9. Soil Type (Clay, Sandy, Loamy)
